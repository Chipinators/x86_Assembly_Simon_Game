To properly run the game properly you MUST keep all files in this folder. 

Simply run the Simon.exe to start the game, Windows may warn you that it is a dangerous file because it is coded in Assembly.

### FIRST TIME LAUNCH SETUP: ###
Upon launching the game for the first time please right click on the header of the window and select "Properties".
Next navigate to the Layouts tab and in the "Window Size" section set the Width to 68 and the Height to 35.
Not setting these window properties will ruin game experience.
